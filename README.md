 <h1 align="center">DevCells Website</h1>
 
 <p align="center">This repository contains source code of <b>DevCells's</b> website</p>
 
 
 ## Building
 
 ```bash
# install dependencies (NPM)
npm i

# install dependencies (Yarn)
yarn

# build website in /dist (NPM)
npm run build

# build website in /dist (Yarn)
yarn build
```

 ## Development
 
 ```bash
# install dependencies (NPM)
npm i

# install dependencies (Yarn)
yarn

# host local server (NPM)
npm run dev

# host local server (Yarn)
yarn dev
```
